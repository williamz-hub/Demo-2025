# Demo-2025

A collection of simple interactive demonstrations.

## Available Demos

### Demo 4: Interactive Counter
- **File**: `demo4-counter.html`
- **Description**: A simple interactive counter application with increment, decrement, and reset functionality
- **Features**:
  - Click buttons to increment (+) or decrement (-) the counter
  - Reset button to return counter to zero
  - Keyboard support (Arrow keys, +/- keys, R for reset)
  - Visual feedback with color changes for positive/negative values
- **Usage**: Open `demo4-counter.html` in a web browser

## How to Run

Simply open any of the HTML demo files in a web browser to try them out.
