# Demo-2025

A collection of demo utilities and tools themed around the year 2025.

## 🎯 2025 Year Facts Utility

An interactive utility that provides interesting facts, historical context, and mathematical properties about the year 2025.

### Features

- **Random Facts**: Get interesting facts about the year 2025
- **Historical Context**: Learn about significant events from past decades
- **Mathematical Properties**: Explore the mathematical aspects of 2025
- **Time Calculations**: See days until/since 2025
- **Dual Interface**: Both command-line and web interfaces available

### Usage

#### Command Line Interface
```bash
python3 year_facts_2025.py
```

#### Web Interface
Open `year_facts_2025.html` in any web browser, or serve it locally:
```bash
python3 -m http.server 8000
# Then visit http://localhost:8000/year_facts_2025.html
```

### Sample Facts

- 2025 is the square of 45 (45² = 2025)
- 2025 marks 30 years since the release of Windows 95
- In Roman numerals, 2025 is written as MMXXV
- 2025 will be 80 years since the end of World War II
