# Demo-2025

A collection of 2025-themed demo utilities and tools.

## Demo 2025 Utility

A simple utility that provides 2025-themed calculations and interesting facts about the year 2025.

### Features

- **Date Calculations**: Calculate days until/since New Year 2025 and days since 2025 started
- **Year Facts**: Display interesting mathematical and cultural facts about 2025
- **Date Day Checker**: Find out what day of the week any date falls on in 2025
- **Dual Interface**: Available as both command-line Python script and web interface

### Usage

#### Python Command Line Interface
```bash
python3 demo2025.py
```

Follow the interactive menu to:
1. Check days until/since New Year 2025
2. See days since 2025 started
3. View interesting facts about 2025
4. Check what day a specific date falls on in 2025

#### Web Interface
Open `demo2025.html` in any web browser for an interactive GUI version with the same functionality.
