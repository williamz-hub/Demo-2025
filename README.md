# Demo-2025

A simple demonstration repository showcasing basic functionality for 2025.

## Features

This repository contains basic demo implementations:

### Python Demo (`demo.py`)
- Interactive command-line demo
- Basic calculator functionality
- User input handling
- Error handling examples

### Web Demo (`index.html`)
- Interactive web-based demo
- Simple calculator with GUI
- Text manipulation tools
- Responsive design

## How to Run

### Python Demo
```bash
python3 demo.py
```

### Web Demo
Open `index.html` in any modern web browser, or serve it with a simple HTTP server:

```bash
# Using Python's built-in server
python3 -m http.server 8000

# Then open http://localhost:8000 in your browser
```
