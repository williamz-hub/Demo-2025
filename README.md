# Demo-2025

Welcome to Demo-2025! This repository contains various demo utilities and applications related to the year 2025.

## 🎯 2025 Countdown Utility

A fun and interactive tool to track time relative to the year 2025, complete with interesting facts about the year.

### Features

- **Days Calculation**: Shows days until 2025 (if before) or days since 2025 began
- **Interactive Facts**: Learn interesting facts about 2025
- **Dual Interface**: Both command-line and web interfaces available
- **Real-time Updates**: Automatically refreshes the countdown
- **Responsive Design**: Works on desktop and mobile devices

### Usage

#### Command Line Interface

Run the Python script directly:

```bash
python3 countdown_2025.py
```

The CLI provides:
- Current countdown information
- Interactive menu to explore facts
- Real-time calculations

#### Web Interface

Open `countdown_2025.html` in any web browser for a beautiful, interactive experience featuring:
- Animated countdown display
- Click to reveal new facts
- Responsive design for all devices
- Auto-refresh functionality

### Files

- `countdown_2025.py` - Python CLI application
- `countdown_2025.html` - Web interface (HTML/CSS/JavaScript)

### Requirements

- Python 3.6+ (for CLI version)
- Any modern web browser (for web version)
- No external dependencies required

### Demo

The utility automatically detects whether we're before, during, or after 2025 and adjusts the display accordingly. It includes fun facts about 2025 such as:

- International Year of Peace and Trust
- World Expo 2025 in Osaka, Japan
- 80th anniversary of WWII's end
- Technological advancement milestones
- Mathematical properties of 2025
